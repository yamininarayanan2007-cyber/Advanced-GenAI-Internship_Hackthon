# Advanced GenAI Internship Hackathon

## Description
This project merges multiple datasets from a food delivery service:
- Orders data (`orders.csv`)
- Users data (`users.json`)
- Restaurants data (`restaurants.sql`)

The Jupyter Notebook `merge_food_delivery_data.ipynb` loads, cleans, and merges these datasets to create a final dataset `final_food_delivery_dataset.csv`.

## How to Use
1. Open `merge_food_delivery_data.ipynb` in Jupyter Notebook or Google Colab.
2. Upload the dataset files: `orders.csv`, `users.json`, `restaurants.sql`.
3. Run all cells to generate `final_food_delivery_dataset.csv`.

## Files in this Repository
- `merge_food_delivery_data.ipynb` → Jupyter Notebook with the complete code
- `orders.csv` → Orders dataset
- `users.json` → Users dataset
- `restaurants.sql` → Restaurants database
- `final_food_delivery_dataset.csv` → Final merged dataset (generated by notebook)

## Notes
- Make sure all dataset files are uploaded before running the notebook.
- Python packages used: `pandas`, `sqlite3`, `json`, `re`.
